# Notebooks d’exemple pour CineInfini

Ce dossier contient des notebooks Jupyter/Colab illustrant l’utilisation du pipeline CineInfini.

## Fichiers

- `01_cineinfini_demo.ipynb` – Démonstration rapide sur une vidéo synthétique.
- `02_tears_of_steel_audit.ipynb` – Audit complet du film libre `Tears of Steel` (live‑action).

## Exécution dans Google Colab

1. Ouvrez le notebook souhaité dans Colab (`Fichier > Ouvrir > GitHub`).
2. Montez votre Google Drive si nécessaire (le code le fait automatiquement).
3. Exécutez les cellules dans l’ordre.

> **Note** : Le téléchargement de `Tears of Steel` (230 Mo) est automatique la première fois.

## Exécution locale

- Installez CineInfini (`pip install cineinfini-audit`).
- Lancez `jupyter notebook` et ouvrez le notebook.
- Modifiez le chemin `base` dans la première cellule si besoin.
